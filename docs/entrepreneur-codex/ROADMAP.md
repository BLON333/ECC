# Entrepreneur Codex Roadmap

This sequence is planning context, not authorization to begin a slice.

1. Foundation and repository memory
2. Operating contracts for Projects, connections, and insurance knowledge
3. Lightweight Entrepreneur ECC profile contract
4. Synthetic and sanitized insurance-assistance pilot
5. Simple Software Studio starter
6. Approved real-data connections and operating controls
7. Evidence-based expansion or consolidation

Stage 4 uses no real client data. Real-data connections and their storage, backup, retention, device security, AI-processing, permission, and operating controls remain a later gate requiring separate approval.
